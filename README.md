# Student Result Management System

## Problem Statement
Develop a Student Result Management System using Python that allows users to store, update, retrieve, and manage student academic records. The system maintains details such as student name, subject-wise marks, total marks, percentage, and grades. This project demonstrates the use of Python programming concepts including loops, conditional statements, lists, dictionaries, and user interaction.

---

## Key Features
- Store student records
- Calculate total marks and percentage
- Generate grades automatically
- Update student details and marks
- Retrieve saved student records
- Menu-driven interface
- Uses Python lists and dictionaries
